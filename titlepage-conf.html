Configuration of the title page

The file config_tpa

This is an example of a configuration file for the titlepage. Some long lines have been split (they start with three dots instead of a line number). The use of this files is described here.

...      'translate also bibliography'"

224   \Dtitle <TeXtitre xml:lang='fr'> "TF"

226   \Btitle <TeXtitre xml:lang='en'> "TE"
227   \currentlanguage <langue> "en"
