2026-02-23, Version 3.0.0

Many changes for this release, mostly none of them affecting the output (too many to list here). Of note: - Switch to CMake for better compatibility - Use typographically correct quotation marks in the output - Change the confdir search algorithm, enable kpathsea lookup for it - Switch to spdlog for logging - Only support UTF8 output
